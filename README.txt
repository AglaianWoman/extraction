==========
Extraction
==========

Extraction is a Python package for extracting titles, descriptions
and images from web pages. You might want to use Extraction if you're
building a link aggregator where users submit links and you want to
display them (like submitting a link to Facebook, Digg or Delicious).


Installation
============

The simplest way to install Extraction is via PyPi::

    pip install extraction

If you want to develop against extraction, you can install from GitHub::

    git clone
    cd extraction
    virtualenv env
    . ./env/bin/activate
    pip install -r requirements.txt
    python setup.py develop

Then you can run the tests::

    python tests.py
